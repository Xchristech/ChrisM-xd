To start the bot, you need a valid session ID.

```js
module.exports = {
  SESSION_ID: process.env.SESSION_ID || 'CHRISM_XD_SESSION_001', // Replace with your real session ID

  BOT_NAME: process.env.BOT_NAME || "CHRISM-XD",
