## ⚙️ How to Get Your Own CHRISM-XD Bot

```bash
git clone https://github.com/Chimbiko-xd/ChrisM-XD
cd ChrisM-XD
npm install

Then set up your .env with:

SESSION_ID=YOUR_SESSION_ID_HERE
BOT_NAME=CHRISM-XD
PREFIX=.
MODE=public
